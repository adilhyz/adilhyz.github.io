CyberFox

Cyberfox merupakan Browser yang berbasis dari Firefox, yang di desain untuk Penetrasi Sistem tersedia di 64-bit dan 32-bit. Diambil dari sumber sourceforge Cyberfox sudah tidak mendapatkan update terbaru lagi sejak 2018. meski begitu Cyberfox tidak kalah unggul dari browser sebelah, karena masih support beberapa ekstensi lawas Yaitu Hackbar, Alat serbaguna untuk Penetrasi Keamanan.

Disini saya sudah menyediakan bahan untuk instalasi, download sesuai asitektur cpu.

CyberFox For Amd 32Bit CyberFox For Intel 32Bit

Linux

Arch based

Arch Btw dikarenakan environment pacman yang mana .zst tidak seperti package .deb dan kita perlu menginstall beberapa mirror tambahan, disini saya sudah menyediakan mirror repositori Cyberfox untuk Arch Linux, Untuk Instalasi saya menyediakan dua Opsi yaitu Otomatis dan Manual

Otomatis Menggunakan echo (Sekali klik di Terminal)

1

sudo echo -e "\n[adilhyz-repo]\nSigLevel = Optional TrustAll\nServer = http://adilhyz.github.io/adilhyz-repo/x86_64\n" | sudo tee -a /etc/pacman.conf > /dev/null

Lalu Update untuk Sinkronkan database pacman

1

sudo pacman -Syyu

Manual edit /etc/pacman.conf

1

sudo vim /etc/pacman.conf

Tambahkan mirror di bagian paling bawah

1
2

SigLevel = Optional TrustAll
Server = http://adilhyz.github.io/adilhyz-repo/x86_64

Lalu Update untuk Sinkronkan database pacman

1

sudo pacman -Syyu

Terakhir Install Cyberfox

1

sudo pacman -S cyberfox
Selesai

Setelah Instalasi CyberFox Menggunakan mirror adilhyz-repo Cyberfox Siap Di Jalankan.

Run CyberFox

Setelah Instalasi CyberFox, untuk pertama kali kita ditampilkan untuk mengkustom tampilan awal, silahkan isi sesuai selera masing-masing. Setelah di pilih, lanjut menginstall Ekstensi terlebih dahulu.

Install Ekstensi Hackbar

Repositori Ekstensi Hackbar Terbaru https://github.com/adilhyz/EXT/archive/refs/tags/1.0.zip

Catatan

Install ekstensi di menu Add-ons, dan Pergi ke menu Extensions.

Impor Backup json

Sesudah Menginstall ekstensi HackBar, kemudian restore backup ke Tools lalu pilih Ikon settings dan file Import File .json

FAQ

Ini bisa dipakai di distro selain Arch?

Tips

Bisa, untuk Fedora, NixOs dan Gentoo. pada saat artikel ini dibuat saya belum mencobanya, intinya jika mencoba dan ngulik setidaknya bisa menurut saya.

Apa perbedaan Cyberfox versi Intel dan AMD?

Catatan

Secara teknis hanya optimisasi build-nya saja yang beda. Versi Intel dan AMD dibangun agar performa maksimal di arsitektur CPU masing-masing. Hingga artikel ini dibuat, dua-duanya bisa jalan di semua CPU.

Di Linux Juga sama itu multifungsi kok, baik intel dan amd bisa berjalan dengan baik.
